## Supported Features

- Controlled/Uncontrolled state management

- Column & row CRUD

- Column & row DND

- Column & row duplicating

- Column wrapping

- Column pinning

- Column type changing

- Cells types: title, text, checkbox

## Installation

```package-install
pnpm add @notion-kit/table-view
```

## API Reference

### TableView

| Prop       | Type                    | Default | Description                                                                          |
| ---------- | ----------------------- | ------- | ------------------------------------------------------------------------------------ |
| `apiKey`\* | `string`                | -       | The Unsplash access key from [Unsplash Developers](https://unsplash.com/developers). |
| `onSelect` | `(url: string) => void` | -       | Handler that is called when an Unsplash image is selected.                           |
