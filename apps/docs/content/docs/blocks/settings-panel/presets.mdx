## Installation

```package-install
pnpm add @notion-kit/settings-panel
```

## Components

---

### Connections Table

A table to display all established connections.

<ComponentPreview
  name="settings-panel/connections-table"
  preview={`<ConnectionsTable data={connections} />`}
/>

#### API Reference

| Prop     | Type                                                             | Default | Description          |
| -------- | ---------------------------------------------------------------- | ------- | -------------------- |
| `data`\* | [`Connection[]`](/docs/blocks/settings-panel/objects#connection) | -       | Current connections. |

### Members Table

A table to display all members (except guests) in the workspace.

<ComponentPreview
  name="settings-panel/members-table"
  preview={`<MembersTable data={data} scopes={scopes} />`}
/>

#### API Reference

| Prop       | Type                                                            | Default | Description                                                    |
| ---------- | --------------------------------------------------------------- | ------- | -------------------------------------------------------------- |
| `scopes`\* | [`Set<Scope>`](/docs/blocks/settings-panel/scopes)              | -       | Current scopes.                                                |
| `data`\*   | [`MemberRow[]`](/docs/blocks/settings-panel/objects#member-row) | -       | Current members and owners.                                    |
| `search`   | `string`                                                        | -       | Current search string.                                         |
| `onUpdate` | `(id: string, role: Role) => void`                              | -       | Handler that is called when the role of the member is changed. |
| `onDelete` | `(id: string) => void`                                          | -       | Handler that is called when the member is removed.             |

### Guests Table

A table to display all guests in the workspace.

<ComponentPreview
  name="settings-panel/guests-table"
  preview={`<GuestsTable data={data} scopes={scopes} />`}
/>

#### API Reference

| Prop       | Type                                                          | Default | Description                                                   |
| ---------- | ------------------------------------------------------------- | ------- | ------------------------------------------------------------- |
| `scopes`\* | [`Set<Scope>`](/docs/blocks/settings-panel/scopes)            | -       | Current scopes.                                               |
| `data`\*   | [`GuestRow[]`](/docs/blocks/settings-panel/objects#guest-row) | -       | Current guests.                                               |
| `search`   | `string`                                                      | -       | Current search string.                                        |
| `onUpdate` | `(id: string, role: Role) => void`                            | -       | Handler that is called when the role of the guest is changed. |
| `onDelete` | `(id: string, name: string) => void`                          | -       | Handler that is called when the guest is removed.             |
