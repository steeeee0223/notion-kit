## API Reference

### Unsplash

| Prop       | Type                    | Default | Description                                                                          |
| ---------- | ----------------------- | ------- | ------------------------------------------------------------------------------------ |
| `apiKey`\* | `string`                | -       | The Unsplash access key from [Unsplash Developers](https://unsplash.com/developers). |
| `onSelect` | `(url: string) => void` | -       | Handler that is called when an Unsplash image is selected.                           |
