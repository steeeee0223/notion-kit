## API Reference

### Unsplash

| Prop       | Type                             | Default | Description                                                                                  |
| ---------- | -------------------------------- | ------- | -------------------------------------------------------------------------------------------- |
| `apiKey`\* | `string`                         | -       | The Unsplash access key from [Unsplash Developers](https://unsplash.com/developers).         |
| `onSelect` | `(image: UnsplashImage) => void` | -       | Handler called when an image is selected. Use `image.urls.small`, `regular`, `full`, or raw. |

### `type` UnsplashImage

```ts
type UnsplashImage = AssetBasic;
```

`UnsplashImage` is exported from `@notion-kit/ui/unsplash` and matches the `unsplash-js` basic asset shape.
