## Create Project

Run the `shadcn init` command to create a new Next.js project or to setup an existing one:

```bash
pnpm dlx shadcn@latest init
```

Choose between a Next.js project or a Monorepo.

## Add Notion Components

You can now start adding Notion components (or blocks) to your project.

```bash
pnpm dlx shadcn@latest https://notion-ui.vercel.app/registry/notion-ui.json
```

The command above will add the Notion primitive UIs to your project. You can then import it like this:

```tsx
import { Button } from "@notion-kit/ui/primitives";

export default function Home() {
  return (
    <div>
      <Button>Click me</Button>
    </div>
  );
}
```

## Manual Setup (Without CLI)

If you are installing Notion Kit packages directly via `npm` rather than the `shadcn` CLI, you must also provide the theme CSS variables for the components to display correctly.

1. Install the config package:

```bash
npm install @notion-kit/tailwind-config
```

2. Add the CSS import to the top of your global stylesheet (e.g., `app/globals.css`):

```css
@import "tailwindcss";
@import "@notion-kit/tailwind-config/base.css";
```
