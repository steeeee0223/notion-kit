## Options

---

### Size

Use the `size` prop to control the size of the switch.
The default size is `"md"`.

<ComponentPreview
  name="switch/sizes"
  preview={`<Switch size="sm"/>
  <Switch size="md"/>
  `}
/>

## API Reference

### Switch

| Prop   | Type           | Default | Description             |
| ------ | -------------- | ------- | ----------------------- |
| `size` | `"sm" \| "md"` | `"md"`  | The size of the switch. |
