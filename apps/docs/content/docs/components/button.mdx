## Options

---

### Variant

Use the `variant` prop to control the visual style of the button.<br/>
The default variant is `"primary"`.

<ComponentPreview
  name="button/variants"
  preview={`<Button variant="primary" size="md">Button</Button>
  <Button variant="hint" size="md">Button</Button>

  <Button variant="soft-blue" size="md">Button</Button>
  <Button variant="red" size="md">Button</Button>
  <Button variant="red-fill" size="md">Button</Button>
  <Button variant="link" size="md">Button</Button>
  `}
/>

## API Reference

### Button

An `Button` extends the HTML `<button>` component. We list the extended properties only.

| Prop      | Type                                                                                                                   | Default     | Description                     |
| --------- | ---------------------------------------------------------------------------------------------------------------------- | ----------- | ------------------------------- |
| `variant` | `"primary" \| "icon" \| "link \| "blue" \| "soft-blue" \| "hint" \| "red" \| "red-fill" \| "white" \| "close" \| null` | `"primary"` | The visual style of the button. |
| `size`    | `"xs" \| "sm" \| "md" \| "lg" \| "circle" \| null`                                                                     | `null`      | The size of the button.         |
