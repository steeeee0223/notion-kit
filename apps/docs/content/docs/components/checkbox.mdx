## Options

### Size

Use the `size` prop to control the size of the checkbox.
The default size is `"md"`.

<ComponentPreview
  name="checkbox/sizes"
  preview={`<Checkbox size="xs"/>
  <Checkbox size="sm"/>
  <Checkbox size="md"/>
  `}
/>

## API Reference

### Checkbox

| Prop   | Type                   | Default | Description               |
| ------ | ---------------------- | ------- | ------------------------- |
| `size` | `"xs" \| "sm" \| "md"` | `"md"`  | The size of the checkbox. |
